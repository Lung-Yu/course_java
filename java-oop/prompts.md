# 物件導向 - 題目

### 題目1：定義一個類別

請設計一個名為 `Student` 的 Java 類別，用來表示一名學生。

**要求：**

1.  **屬性 (Attributes)**：
    *   `name`：學生的姓名 (型別為 `String`)。
    *   `score`：學生的分數 (型別為 `int`)。

2.  **方法 (Method)**：
    *   `displayInfo()`：一個公開 (public) 的方法，沒有回傳值 (void)，功能是在控制台印出學生的姓名與分數，格式為 `"姓名: [name], 分數: [score]"`。
