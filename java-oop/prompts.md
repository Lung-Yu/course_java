# Java 物件導向程式設計 - 課程題目

## 課程概覽

本課程涵蓋Java物件導向程式設計的核心概念，從基礎的OOP原則到進階的設計模式與現代Java特性。

---

## 1. 物件導向設計原則

### 題目1-1：理解物件導向概念
設計一個簡單的 `Book` 類別來展示物件導向的基本概念。

**要求：**
- 包含書名、作者、價格等屬性
- 實現封裝：所有屬性為private，提供getter/setter方法
- 提供顯示書籍資訊的方法

### 題目1-2：OOP四大原則展示
創建一個動物類別系統，展示封裝、繼承、多型、抽象化原則。

---

## 2. 類別設計與建構

### 題目2-1：基本Student類別
請設計一個名為 `Student` 的 Java 類別，用來表示一名學生。

**要求：**
1. **屬性 (Attributes)**：
   - `name`：學生的姓名 (型別為 `String`)
   - `score`：學生的分數 (型別為 `int`)

2. **方法 (Method)**：
   - `displayInfo()`：顯示學生資訊的方法

### 題目2-2：多建構子設計
擴展Student類別，加入多個建構子和更多功能。

**要求：**
- 預設建構子
- 帶參數的建構子
- 複製建構子
- 方法重載示例

### 題目2-3：封裝與存取修飾符
設計一個 `BankAccount` 類別，展示封裝的重要性。

---

## 編譯與執行指令

```bash
# 編譯Java檔案
javac ClassName.java

# 執行Java程式
java ClassName

# 編譯多個檔案
javac *.java

# 執行包含main方法的類別
java MainClassName
```
