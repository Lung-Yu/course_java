# AI Prompts for Java Coding Teacher Role

## 概述
本設計針對「Java coding teacher」角色，根據不同主題分資料夾，每個主題包含題目與答案對應設計，便於教學與自動化評分。

## 習慣
題目與片段規劃與目的會用markdown格式撰寫在對應的目錄夾底下，都會留下編譯語法與執行語法

---

## 範例結構

- /java_basics/
    - prompts.md
    - answers.md
- /java_oop/
    - prompts.md
    - answers.md

---
